# ============================================================================
# PACKAGE EXPLANATIONS:
# ============================================================================
#
# Flask:
# - Micro web framework for Python
# - Handles routing, requests, and responses
# - Provides templating with Jinja2
# - Built-in development server
#
# Flask-SQLAlchemy:
# - Database toolkit for Flask
# - Object Relational Mapper (ORM)
# - Allows working with databases using Python classes
# - Supports SQLite, PostgreSQL, MySQL, etc.
#
# Flask-WTF:
# - Integration of WTForms with Flask
# - Provides CSRF protection
# - Makes form handling secure and easy
# - Automatic form rendering in templates
#
# WTForms:
# - Form library for Python
# - Provides form fields and validators
# - Handles form data and validation
# - Generates HTML form fields
#
# Email-validator:
# - Validates email addresses
# - Checks email format and DNS records
# - Required by WTForms EmailField
# - Prevents invalid email addresses
#
# Werkzeug:
# - WSGI utility library
# - Provides security helpers (password hashing)
# - URL routing utilities
# - Debugging tools
#
